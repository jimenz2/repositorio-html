Glosario de HTML y CSS

Guía completa de etiquetas básicas para estructurar páginas web

Estructura Básica Etiquetas de Texto Contenedores Formularios CSS Básico

Estructura Básica HTML

Documento

<!DOCTYPE html>

Define que el documento es HTML5

<!DOCTYPE html>
Raíz

<html>

Elemento raíz de un documento HTML

<html lang="es">...</html>
Metadatos

<head>

Contiene metadatos y enlaces a recursos

<head> <meta charset="UTF-8"> <title>Título de la página</title> </head>
Contenido

<body>

Contiene el contenido visible de la página

<body>...</body>

Etiquetas de Texto

Encabezados

<h1> a <h6>

Define encabezados de diferentes niveles

<h1>Título Principal</h1> <h2>Subtítulo</h2>
Párrafo

<p>

Define un párrafo de texto

<p>Este es un párrafo</p>

Propiedades CSS Básicas

Estilo

color

Define el color del texto

color: #ff0000; color: rgb(255, 0, 0);
Diseño

margin / padding

Define los márgenes externos e internos

margin: 10px; padding: 15px 20px;
